Minecraft Servers

I host minecraft servers for my friends, and have been doing so ever since Secondary 1. I use a variety of different methods, including using Spigot, a very popular 3rd-party minecraft server and Vanilla, which is provided by Microsoft through Mojang. I have hosted them in different envrionments, some of which are Windows, which a server is currently running on, Linux on a dedicated server and macOS in the earlier days. Join us!

Computer Building

Other than software, I love doing hardware projects as well. For example, the mentioned minecraft server is built by yours truly. To build my computers, I do thorough research on whether the parts I pick are compatible with one another. To do that, I google the information or use PC Part Picker, which is a really useful website that guides you along the building process and tells you whether your parts are compatible or not.
